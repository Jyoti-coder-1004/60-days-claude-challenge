# Day 10: Building a Personal Portfolio Website

## 🚀 Overview
Today, I built and designed a modern, responsive, single-file personal portfolio website using HTML, Tailwind CSS, and vanilla JavaScript. The goal was to transform my resume into a visually compelling, interactive personal brand showcase.

---

## 🛠️ Tech Stack Used
- **Markup:** HTML5
- **Styling:** Tailwind CSS (via CDN)
- **Interactivity:** Vanilla JavaScript
- **AI Tooling:** Claude AI (Prompt engineering for component generation)

---

## ✨ Features Implemented
- **Hero Section:** Clean intro featuring my name, professional title, a smooth typing animation effect, and social links.
- **About Me:** A recruiter-focused summary highlighting my background and aspirations.
- **Skills Matrix:** Dynamic technical skill tags combined with animated proficiency bars.
- **Projects Grid:** Modern, responsive project cards highlighting tech stacks and core features.
- **Achievements & Certifications:** A structured timeline highlighting key milestones.
- **Dark/Light Mode:** A fully operational dark/light toggle with persisting user preferences.
- **Responsiveness:** Fluid styling optimized for mobile, tablet, and desktop views.

---

## 💡 Key Learnings
1. **Prompt Structuring:** Learned how providing structured markdown inputs (`=== PERSONAL INFO ===`, `=== REQUIREMENTS ===`) helps LLMs generate complete, single-file codebases without omitting essential script blocks.
2. **Tailwind via CDN:** Understood how to build quick, production-ready interfaces rapidly without setting up complex local build steps (like Webpack or Vite) for simple static sites.
3. **UI/UX Consistency:** Learned the importance of uniform color spacing, smooth-scroll anchoring, and subtle hover animations to make a web page feel premium.
