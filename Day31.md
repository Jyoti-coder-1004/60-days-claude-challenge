# Day 31: AI Supply Chain Control Tower Simulation

## 📌 Project Overview
For Day 31 of the 60-Day Coding Challenge, I built a complete, interactive **AI Supply Chain Control Tower** application. The system replicates the high-stakes environment of a global Head of Operations, managing live logistical disruptions in real-time.

The entire application is engineered as a single, self-contained HTML file utilizing vanilla web technologies without any heavy frameworks or external dependencies.

---

## 🛠️ Key Learnings

### 1. Control Tower Architecture
* Gained a deep understanding of how enterprise operations monitor real-time supply chains.
* Learned how to structurally balance interconnected business metrics like **Service Level %, Customer Satisfaction, Inventory Health,** and **Operating Costs**.

### 2. Operational Decision-Making
* Mastered prioritizing critical incoming alerts (e.g., Port Congestion, Supplier Delays, Factory Machine Failures) based on real-world urgency and remaining time limits.
* Analyzed the immediate and delayed financial consequences of choices like *Expediting Shipments* versus *Using Backup Suppliers*.

### 3. Core Technical Skills
* Built a premium, dark-themed operations dashboard with smooth CSS transitions and real-time state management in JavaScript.
* Implemented modular game logic featuring dynamic scaling difficulty, countdown timers, and live event logging.
