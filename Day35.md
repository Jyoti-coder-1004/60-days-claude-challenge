# Day 35: Interactive Prompt Engineering — Prompt Puzzle

## 📌 Project Overview
Built a premium, single-file interactive educational game called **🧩 Prompt Puzzle — Master AI Prompting Through Play**. The application helps users master the core principles of prompt engineering (role, context, constraints, and formatting) through gamified challenges rather than traditional passive reading.

## 🧠 Key Learnings
1. **Structural Scaffolding:** Learned how breaking prompts down into modular, draggable blocks helps users visualize the anatomy of a perfect AI prompt.
2. **Optimization Dynamics:** Recognized the thin line between a weak, an optimized, and an over-engineered prompt, proving that more text does not always equal better AI outputs.
3. **Interactive Instructional Design:** Understood how implementing live scoring mechanics (Accuracy, Moves, Wrong Placements, Hints) dramatically improves user engagement and retention.

## 🚀 Game Features Built
- **Three Challenge Types:** 
  1. *Build the Prompt* (Drag-and-drop structural elements)
  2. *Clean the Prompt* (Remove redundant and vague components)
  3. *Choose the Best Prompt* (Identify the most optimized variation)
- **Live Scoring Engine:** Metrics tracking accuracy, time, moves, and hints used to calculate an overall optimization bonus.
- **Prompt Performance Report:** Generates a personalized final breakdown featuring a unique "Prompt DNA" visualization, rank, and custom feedback.
- **Offline Reliability:** Built cleanly using vanilla HTML, CSS, and JavaScript wrapped seamlessly inside one file.
