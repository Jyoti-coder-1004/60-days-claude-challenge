# Day 42 – Personal Financial Command Center

## 📌 Project Overview

**Personal Financial Command Center** is a premium single-page financial dashboard designed to help users understand, manage, and improve their financial health. Instead of acting as a simple expense tracker, it provides an intelligent financial command center with interactive analytics, budgeting tools, financial planning, goal tracking, and AI-powered recommendations.

The application begins with a guided interview that personalizes the experience according to the user's financial profile. For this version, the selected profile is **Student**, resulting in a dashboard focused on student budgeting, savings, education expenses, and financial literacy.

---

# 🎯 Objectives

* Help users understand their complete financial situation.
* Encourage healthy budgeting and saving habits.
* Visualize financial data through interactive dashboards.
* Provide personalized recommendations.
* Improve financial literacy through practical insights.
* Support long-term financial planning.

---

# 👤 Target User

**Student**

The dashboard is optimized for students managing:

* Pocket money

* Transportation
* Online subscriptions
* Emergency savings
* Education-related spending

---

# 🧠 Personalization Flow

Before generating the dashboard, the application asks one question at a time using MCQs:

1. User Type
2. Student financial profile
3. Income source
4. Monthly budget range
5. Financial goals
6. Dashboard customization preferences

This creates a personalized experience instead of a generic finance application.
