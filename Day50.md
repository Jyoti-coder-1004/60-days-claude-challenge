# Day 50: Build Defend Your Experience

**Goal:** Create an AI-Powered Adaptive Interview Defense Simulator.  
**Level:** Advanced  
**Estimated Time:** 120 min  
**Deliverable:** GitHub commit URL  

---

## 🎯 Task Overview

Build a single, self-contained interactive web application using HTML, CSS, and JavaScript that helps users prepare for tough interviews. Instead of just reviewing resumes, the application extracts claims from uploaded documents (resumes, portfolios, research, etc.) and acts as an intelligent skeptic—continuously challenging users with personalized follow-up questions to help them confidently defend every claim they make.
