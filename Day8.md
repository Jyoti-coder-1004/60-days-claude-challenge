# Day 8: Building a Personal Environmental Health Analyzer Dashboard

## 🎯 Project Overview
For Day 8 of the challenge, I transformed Claude from a text-based AI assistant into a full-stack application builder. Using advanced prompting techniques, I instructed Claude to act as a Senior Data Analyst, UX Designer, and Frontend Developer to create a fully interactive **Personal Environmental Health Analyzer Dashboard**.

---

## 📸 Dashboard Preview & Features

### 1. Executive Summary & Key Metrics
The dashboard aggregates real-time AQI and water quality data to calculate an overall Environmental Health Score. 

![Executive Summary & Metrics](placeholder-for-your-screenshot1.png)
*Features: Live widgets displaying Average AQI, Highest/Lowest AQI cities, and individual Air/Water Quality letter grades.*

### 2. Interactive Visualizations & Filters
Users can actively slice the data using dynamic controls to spot environmental trends.

![Visualizations & Filters](placeholder-for-your-screenshot2.png)
*Features: Interactive pollutant selectors (PM2.5 vs PM10), city comparison modes, and beautifully rendered dark-themed charts.*

### 3. Personalized Health Insights & Action Plan
The app maps environmental data directly to personal wellness markers (skin, hair, and respiratory health) and provides custom recommendations.

![Health Insights Panel](placeholder-for-your-screenshot3.png)
*Features: Dynamic risk indicators (🟢 Low, 🟡 Moderate, 🔴 High) and tailor-made indoor/outdoor guidance.*

---

## 💡 Key Learnings & Takeaways

* **Artifacts as App Builders:** I learned how to move beyond basic chatbot interactions and utilize Claude Artifacts to generate self-contained, downloadable HTML applications that run locally with fully functional JavaScript filters.
* **UX/UI Structuring via Prompts:** Witnessed how explicitly defining a design language (e.g., "dark theme, modern premium UI, mobile responsive") changes how AI generates frontend code, resulting in production-ready CSS.
* **Component Integration:** Observed how seamlessly Claude combined completely distinct data domains—mapping environmental metrics (AQI/Water hardness) directly into actionable biological risks (hair fall, acne, lung performance).
